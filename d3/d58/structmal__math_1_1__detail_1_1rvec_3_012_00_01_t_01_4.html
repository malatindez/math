Detailed Description

template<Primitive T>
struct mal_math::_detail::rvec< 2, T >

Specialization of rvec for 2D vectors.

Template Parameters
TUnderlying primitive type.

The rvec<2, T> structure represents a 2-dimensional vector with reference-like semantics. This is particularly useful in situations where you want to treat parts of memory as vectors without copying.
